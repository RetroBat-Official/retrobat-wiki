Main Dolphin branch (and libretro-core) have integrated Triforce emulation support since february 2026.

## Controls

Controls are similar to Gamecube mapping for Dolphin.

You can create your own mapping directly in the dolphin-triforce emulator, using the same procedure as for Gamecube.

| RetroBat                                                                       | TriForce          |
| ------------------------------------------------------------------------------ | ----------------- |
| START                                                                          | START             |
| SELECT                                                                         | COIN              |
| D-PAD                                                                          | D-PAD             |
| Left analog stick                                                              | Left analog stick |
| Right analog stick                                                             | C-Stick           |

Service Menu can be accessed from your controller using R3 and L3 buttons or from the keyboard using 0 and 9 keys.
